在 AWS AutoScaling Group 中替换 Instance

Sat, Aug 2, 2014 in DevOps using tags DevOps , Python , Boto , AWS

这两周，我被分配的任务是实现 AWS 的 Auto Scaling功能。多亏有了 Boto， 很快就实现了创建 Auto Scaling Group 和添加 Scaling Policy。但是有一个问题却花费了一些时间才顺利解决。

我们的团队每周四发布新的代码。为了确保服务不中断，更新代码的步骤如下：

  1. 创建并配置（下载新发布的代码）多个新的 Instance；
  2. 依次关闭旧的 Instance，每关闭一个 Instance，就激活（运行代码）一个新的 Instance 来替代被关闭的 Instance。

这样可以保证在 AWS 上运行的 Instance 依次得到更新，并且服务没有中断。

现在的问题是：使用了 Auto Scaling 之后，每次尝试改变 Auto Scaling Group 当中的 Instance 数目，都会激发 Scaling Policy。比如我想关闭一个旧的 Instance，这样 Auto Scaling Group 当中 Instance 的数量就会小于 Desired Capacity，Auto Scaling Group 就会新创建一个 Instance， 而不是等我激活一个 Instance 去替代。

我想到的办法步骤如下：

  1. 将 Auto Scaling Group 挂起（避免激发 Scaling Policy）；
  2. 按照以前的方法替换旧的 Instance；
  3. 将新创建的 Instance 添加到 Auto Scaling Group。

因为 AWS Auto Scaling 的特性（比如 Auto Scaling Group 被挂起的时候不能够添加 Instance，desired_capacity 不能小于 min_size），实现第三步并不容易。具体来说第三步又要分成下面几个步骤：

  1. 将 Auto Scaling Group 的 desired_capacity 和 min_size 分别置为 0（防止恢复 Auto Scaling Group 的时候自动创建 Instance）；
  2. 恢复 Auto Scaling Group（结束挂起状态，然后才可以添加 Instance）；
  3. 添加 Instance 到 Auto Scaling Group 里；
  4. 恢复 Auto Scaling Group 的 min_size。

下面提供一份示例代码以供参考：

  1#!python
  2import sys
  3import time
  4
  5from boto.ec2.autoscale import AutoScaleConnection
  6from boto.ec2.connection import EC2Connection
  7
  8AUTO_SCALING_GROUP = "test_auto_scaling_group"
  9NEW_INSTANCES = ["i-4acb0666", "i-4acbqa9"]
 10CHECK_INTERVAL = 5
 11
 12
 13def suspend(as_conn):
 14    """
 15    Suspend auto scaling group
 16    """
 17    as_conn.suspend_processes(AUTO_SCALING_GROUP)
 18    print "Suspend Done\n"
 19
 20
 21def resume(as_conn):
 22    """
 23    Resume auto scaling group
 24    """
 25    as_conn.resume_processes(AUTO_SCALING_GROUP)
 26    print "Resume Done\n"
 27
 28
 29def get_instance_by_id(ec2conn, inst_id):
 30    """
 31    Get instance by its id
 32    """
 33    ...
 34    pass
 35
 36def deactivate_instance(inst):
 37    """
 38    Deactivate instance
 39    """
 40    ...
 41    pass
 42
 43
 44def delete_instances(as_conn, ec2conn):
 45    """
 46    Delete instances in auto scaling group
 47    """
 48    as_group = as_conn.get_all_groups(names=[AUTO_SCALING_GROUP])[0]
 49    instances = [get_instance_by_id(ec2conn, i.instance_id) for i in as_group.instances]
 50    for instance in instances:
 51        if instance.state != "terminated":
 52            print "*** Terminate Instance %s" % instance.id
 53            deactivate_instance(instance)
 54    # All instances should be terminated
 55    while any([inst.update() != "terminated" for inst in instances]):
 56        sys.stdout.write(".")
 57        sys.stdout.flush()
 58        time.sleep(CHECK_INTERVAL)
 59    print ".\nDelete Done\n"
 60
 61
 62def attach_instances(as_conn):
 63    """
 64    Attach instances into auto scaling group
 65    """
 66    while True and NEW_INSTANCES:
 67        as_group = as_conn.get_all_groups(names=[AUTO_SCALING_GROUP])[0]
 68        if len(as_group.instances) == 0:
 69            as_conn.attach_instances(AUTO_SCALING_GROUP, NEW_INSTANCES)
 70            print "Attach Done\n"
 71            return
 72        else:
 73            print "There are instances still in the group"
 74            time.sleep(CHECK_INTERVAL)
 75
 76
 77def set_capacity(as_conn):
 78    """
 79    Set desired capacity and minimum size as 0
 80    """
 81    as_group = as_conn.get_all_groups(names=[AUTO_SCALING_GROUP])[0]
 82    as_group.min_size = 0
 83    as_group.desired_capacity = 0
 84    as_group.update()
 85    print "Set Capacity Done\n"
 86
 87
 88def resume_capacity(as_conn):
 89    """
 90    Resume minimum size as the original value
 91    """
 92    as_group = as_conn.get_all_groups(names=[AUTO_SCALING_GROUP])[0]
 93    setattr(as_group,"min_size", 1)
 94    as_group.update()
 95    print "Resume Capacity Done\n"
 96
 97
 98def main():
 99    as_conn = AutoScaleConnection()
100    ec2conn = EC2Connection()
101
102    print "Suspend Auto Scaling Group"
103    suspend(as_conn)
104
105    print "Delete Instances"
106    delete_instances(as_conn, ec2conn)
107
108    print "Set Capacity"
109    set_capacity(as_conn)
110
111    print "Resume Auto Scaling Group"
112    resume(as_conn)
113
114    print "Attach Instances"
115    attach_instances(as_conn)
116
117    print "Resume Capacity"
118    resume_capacity(as_conn)
119
120
121if __name__ == "__main__":
122    main()
