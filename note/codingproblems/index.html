Two Coding Problems

Fri, Dec 18, 2015 in Miscellaneous using tags Python , Interview

Two coding problems I encountered in a recent interview.

Problem 1

Problem: Print a binary tree row-by-row, from top to bottom.

Example:

1Input: Binary tree
2  3
3 / \
41   4
5 \   \
6  2   5
7  
8Output: 3, 1, 4, 2, 5

Solution

 1class Node(object):
 2    def __init__(self, value, left=None, right=None):
 3        self.value = value
 4        self.left = left
 5        self.right = right
 6
 7
 8def traverse(rootnode):
 9    # The list to save all visited nodes
10    visited = list()
11
12    # BFS
13    thislevel = [rootnode]
14    while thislevel:
15        nextlevel = list()
16        for n in thislevel:
17            visited.append(n.value)
18            if n.left:
19                nextlevel.append(n.left)
20            if n.right:
21                nextlevel.append(n.right)
22
23        thislevel = nextlevel
24    # Print
25    print(', '.join(map(str, visited)))

Output for the given example

1t = Node(3, Node(1, right=Node(2)), Node(4, right=Node(5)))
2traverse(t)

Problem 2

Problem: Write a function to add two numbers represented as arrays of digits.

Example: [1, 2, 3] + [3, 2, 1] = [4, 4, 4] => 123 + 321 = 444

The given example is confusing. There are two difficulties:

  1. carry digit when adding two digits, e.g. [1, 2, 3] + [9, 8, 7];
  2. input two arrays of different size, e.g. [1, 2, 3] + [4, 3, 2, 1]

Although and the interviewer didn’t mention those points, I thought I should consider them. Otherwise, the problem can be easily solved by a loop or one line code of python [x + y for x, y in zip(lst1, lst2)].

Similar problem solved by C++: Add two numbers represented by linked lists.

Solution

 1def add(lst1, lst2):
 2    ret = list()
 3    # minimum size
 4    length = min(len(lst1), len(lst2))
 5
 6    carry = 0
 7    # From left to right
 8    for i in range(-1, -(length + 1), -1):
 9        ret.append((carry + lst1[i] + lst2[i]) % 10)
10        carry = (carry + lst1[i] + lst2[i]) / 10
11
12    # Adding extra digits
13    if len(lst1) > length:
14        for n in lst1[len(lst1) - length - 1::-1]:
15            ret.append((carry + n) % 10)
16            carry = (carry + n) / 10
17
18    if len(lst2) > length:
19        for n in lst2[len(lst2) - length - 1::-1]:
20            ret.append((carry + n) % 10)
21            carry = (carry + n) / 10
22
23    if carry > 0:
24        ret.append(carry)
25
26    # Reverse result list
27    return ret[::-1]
